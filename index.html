🔥 Firebase 설정

실시간 공유를 위해 Firebase 연결이 필요해요. 아래 순서대로 진행해주세요.

1단계 — console.firebase.google.com 접속
→ 프로젝트 만들기 클릭 → 이름 입력 (예: mevalotin-dashboard)
→ Google Analytics 사용 안함 선택 → 프로젝트 만들기
2단계 — 프로젝트 홈 → Realtime Database → 데이터베이스 만들기
→ 위치: asia-southeast1 → 테스트 모드로 시작
3단계 — 프로젝트 설정(⚙️) → 일반 탭 → 하단 웹 앱 추가(</>)
→ 앱 이름 입력 → firebaseConfig 값 전체 복사
